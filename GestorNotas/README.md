# GestorNotas

## Nombre del alumno
Nicole Peredo Vásquez

## Descripción
Aplicación Java de consola para gestionar notas de texto.
Permite crear notas, listarlas y filtrar las importantes.

## Ramas creadas
- main
- feature-estructura
- feature-alta-notas
- feature-validacion-alta
- feature-listado
- feature-filtro-importancia

## Conflicto resuelto
Se provocó un conflicto en Main.java modificando la cabecera
del menú en main y en feature-listado de forma diferente.
Se resolvió manualmente dejando la cabecera:
=== GESTOR DE NOTAS - VERSION FINAL ===

## Funcionalidad final implementada
- Creación de notas con título, contenido e importancia
- Validación de datos al crear notas
- Listado completo de notas
- Listado filtrado de notas importantes
